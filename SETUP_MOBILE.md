# 📱 Инструкция по установке КиберЩит на телефон

---

## 🤖 ANDROID

### Способ А — Установить как приложение прямо с сайта (проще всего, без компьютера)

1. Откройте сайт КиберЩит на телефоне в браузере **Google Chrome**
2. Нажмите **⋮** (три точки) в правом верхнем углу браузера
3. Выберите пункт **«Добавить на главный экран»** или **«Установить приложение»**
4. Нажмите **«Установить»** в появившемся диалоге
5. Готово — иконка КиберЩит появится на рабочем столе

> ⚠️ Если пункта «Добавить на главный экран» нет — попробуйте Samsung Internet или обновите Chrome.

### Способ Б — Скачать APK файл через GitHub Actions (нужен компьютер)

#### Шаг 1 — Подключите GitHub
В редакторе poehali.dev: **Скачать → Подключить GitHub**
- Авторизуйтесь через GitHub
- Выберите аккаунт, код уйдёт в репозиторий

#### Шаг 2 — Дождитесь сборки
1. Откройте ваш репозиторий на github.com
2. Перейдите во вкладку **Actions**
3. Найдите workflow **«Build Android APK»**
4. Дождитесь зелёной галочки ✅ (занимает 5–10 минут)

#### Шаг 3 — Скачайте APK
1. Нажмите на завершённую сборку
2. Прокрутите вниз до раздела **Artifacts**
3. Нажмите **«kibershield-debug.apk»** — файл скачается на компьютер

#### Шаг 4 — Перенесите APK на телефон
- Отправьте файл себе в **Telegram** или **WhatsApp**
- Или подключите телефон к компьютеру через USB и скопируйте файл

#### Шаг 5 — Разрешите установку из неизвестных источников
1. Откройте файл APK на телефоне
2. Появится предупреждение — нажмите **«Настройки»**
3. Включите **«Разрешить установку из этого источника»**
4. Вернитесь назад и нажмите **«Установить»**
5. Готово! 🎉

### Способ В — Подписанный APK для Google Play (для публичной публикации)

Потребуется создать keystore-файл (цифровая подпись):

#### На компьютере (нужна Java):
```bash
keytool -genkey -v \
  -keystore kibershield.keystore \
  -alias kibershield \
  -keyalg RSA \
  -keysize 2048 \
  -validity 10000
```
Запомните все пароли, которые введёте!

#### Добавьте секреты в GitHub:
Репозиторий → **Settings → Secrets and variables → Actions → New repository secret**

| Название секрета | Значение |
|-----------------|----------|
| `ANDROID_KEYSTORE_BASE64` | Выполните: `base64 -w 0 kibershield.keystore` и вставьте результат |
| `ANDROID_KEY_ALIAS` | `kibershield` |
| `ANDROID_KEY_PASSWORD` | Пароль ключа, который вводили |
| `ANDROID_STORE_PASSWORD` | Пароль хранилища, который вводили |

После добавления секретов запустите сборку повторно — в Artifacts появится `kibershield-release.apk`.

## 🍎 iOS (iPhone / iPad)

> ⚠️ Для iOS обязательно нужен **платный аккаунт разработчика Apple** ($99/год) и **компьютер Mac**.
> Без этого установить на реальное устройство невозможно — такова политика Apple.

### Шаг 1 — Зарегистрируйте аккаунт разработчика
1. Перейдите на [developer.apple.com](https://developer.apple.com)
2. Нажмите **«Account»** → войдите через Apple ID
3. Оплатите подписку **Apple Developer Program** ($99/год)

### Шаг 2 — Создайте App ID
1. В личном кабинете разработчика → **Certificates, IDs & Profiles**
2. Раздел **Identifiers** → нажмите **«+»**
3. Выберите **App IDs** → **App**
4. В поле Bundle ID введите: `ru.kibershield.app`
5. Нажмите **Continue** → **Register**

### Шаг 3 — Создайте сертификат Distribution
1. Раздел **Certificates** → нажмите **«+»**
2. Выберите **Apple Distribution**
3. Создайте CSR-файл на Mac: Приложение **Keychain Access → Certificate Assistant → Request a Certificate from a Certificate Authority**
4. Загрузите CSR, скачайте готовый `.cer`, дважды кликните для установки в Keychain

### Шаг 4 — Узнайте UDID тестового iPhone
1. Подключите iPhone к Mac через USB
2. Откройте **Finder** → нажмите на iPhone
3. Нажмите на строку с моделью — появится UDID
4. Скопируйте UDID

### Шаг 5 — Создайте Provisioning Profile
1. Раздел **Profiles** → нажмите **«+»**
2. Выберите **Ad Hoc** (для тестирования на конкретных устройствах)
3. Выберите созданный App ID и сертификат
4. Добавьте UDID тестового устройства
5. Скачайте файл `.mobileprovision`

### Шаг 6 — Экспортируйте сертификат в .p12
1. Откройте **Keychain Access** на Mac
2. Найдите ваш Distribution сертификат
3. Правая кнопка → **Export** → выберите формат **.p12**
4. Задайте пароль и сохраните файл

### Шаг 7 — Добавьте секреты в GitHub
Репозиторий → **Settings → Secrets and variables → Actions**

| Название секрета | Как получить |
|-----------------|--------------|
| `APPLE_CERTIFICATE_BASE64` | В терминале Mac: `base64 -i certificate.p12 \| pbcopy` |
| `APPLE_CERTIFICATE_PASSWORD` | Пароль, заданный при экспорте .p12 |
| `APPLE_PROVISIONING_PROFILE_BASE64` | `base64 -i profile.mobileprovision \| pbcopy` |
| `KEYCHAIN_PASSWORD` | Придумайте любой пароль |

### Шаг 8 — Установите IPA на iPhone
1. После сборки скачайте `.ipa` из **Actions → Artifacts**
2. Установите через **Apple Configurator 2** (бесплатно в App Store на Mac)
3. Подключите iPhone → перетащите .ipa в приложение
4. На iPhone: **Настройки → Основные → VPN и управление устройством** → доверяйте сертификату

## ❓ Частые вопросы

**Q: Какой способ самый простой для Android?**
A: Способ А — добавление на главный экран прямо из браузера. Занимает 30 секунд.

**Q: APK — это безопасно?**
A: Да, это ваш собственный код, собранный из вашего репозитория.

**Q: Можно ли опубликовать в Google Play?**
A: Да, для этого нужен аккаунт разработчика Google ($25 единоразово) и подписанный APK (Способ В).

**Q: Можно ли опубликовать в App Store?**
A: Да, но нужен аккаунт разработчика Apple ($99/год) и прохождение модерации Apple.
